## AI-Powered FAQ Chatbot for Ed-Tech Platforms Using RAG and Semantic Search
---
#### Tools used:
- LangChain
- Prompt Chaining
- FAISS & Pinecone vector DB
- HuggingFace Embeddings
- Groq
---
This is an AI-powered FAQ chatbot designed to answer queries related to various courses offered by an Ed-Tech platform. The application is built using a public dataset in CSV format containing course-related information and frequently asked questions. The system generates embeddings using Hugging Face embedding models and stores them in Pinecone, a cloud-based vector database. It then performs semantic similarity search on the stored embeddings to retrieve the most relevant context, enabling accurate, context-aware, and intelligent responses to user queries.
